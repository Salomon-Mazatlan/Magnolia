# Supported document types and formats

| Type   | Supported formats                                   |
| ------ | --------------------------------------------------- |
| Text   | `.txt .md .markdown .pdf .docx .rtf .odt`           |
| Image  | `.jpg .jpeg .png .gif .webp .tif .tiff .heic .heif` |
| Video  | `.mp4 .mov .avi`                                    |
| Audio  | `.wav .mp3 .ogg .flac .m4a .aac`                    |
| Survey | `.csv .xlsx`                                        |

::: info

`.docx` and `.rtf` files are converted to `.pdf` when imported.

:::
