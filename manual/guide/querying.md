# Querying

Magnolia's graphical query builder lets you interrogate your data by dragging
code nodes and logical operators onto a canvas.

> Stub page — replace with your real instructions.

## Building a query

1. Open the query builder.
2. Drag code nodes onto the canvas.
3. Connect them with logical operators (AND, OR, NOT, NEAR…).
4. Results update in real time — even while you're still coding.

## Scoping a query

You can scope any query to specific documents, folders, types, or tags.

## Saving queries

Save a query and re-run it whenever your data changes.
