# Coding your data

Coding is the heart of qualitative analysis: you mark passages of your data and
attach a label — a *code* — to them. Think of it as having an infinite supply of
highlighters, where each highlighter has a name.

> This page is a placeholder showing how a content page looks. Replace the text
> below with your real instructions, and add images as shown.

## Applying a code

1. Open a document in the main pane.
2. Select the passage you want to code.
3. Choose an existing code, or create a new one.

## Adding images

Drop image files into `manual-src/guide/` (or an `images/` folder) and reference
them with standard markdown:

```markdown
![Magnolia's coding interface](./images/coding.png)
```

## Tips, warnings, and notes

VitePress gives you styled callout boxes:

::: tip
Codes can be organised into a hierarchy — drag one code onto another to nest it.
:::

::: warning
Deleting a code removes it from every passage it was applied to.
:::
